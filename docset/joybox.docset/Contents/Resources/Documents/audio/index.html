SOUNDS & MUSIC

Audio

Provides support methods for controlling the audio playback of the Game.

Level Up!

You can mute the BackgroundAudio and AudioEffects of the all game using the Audio mute method.

Mute

Mutes all the game audio:

Audio.mute
More intel!

It does not stop the audio playback on the BackgroundAudio and AudioEffect class.

Unmute

Resumes the audio reproduction:

Audio.unmute

Muted?

Returns true if the audio playback is muted:

Audio.muted?

Enable

Enables the audio reproduction:

Audio.enable

Disable

Disables all the game audio:

Audio.disable
More intel!

It stops the audio playback on the BackgroundAudio and AudioEffect class.

Enabled?

Returns true if the audio reproduction is enabled:

Audio.enabled?
