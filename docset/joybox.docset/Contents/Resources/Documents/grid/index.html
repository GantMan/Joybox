ACTIONS & ANIMATIONS

Grid

The Grid class . . .

Flip X 3D

CCFlipX3D action:

flip_x_3d_action = Grid.flip_x_3D

Options

duration: - Float - Default: 0.3
Sets the total duration time of the Action.

Flip Y 3D

CCFlipY3D action:

flip_y_3d_action = Grid.flip_y_3D

Options

duration: - Float - Default: 0.3
Sets the total duration time of the Action.

Lens 3D

CCLens3D action:

lens_3d_action = Grid.lens_3D grid:[100, 200]

Options

position: - CGPoint - Default: [0, 0]
Sets the center position of the grid.
radius: - Float - Default: 0.0
Sets the radius.
duration: - Float - Default: 0.3
Sets the total duration time of the Action.

Waves

CCWaves action:

waves_action = Grid.waves grid:[100, 200]

Options

waves: - Integer - Default: 0
Sets the waves.
amplitude: - Float - Default: 0.0
Sets the amplitude of the waves.
horizontal: - Boolean - Default: false
Enables the horizontal.
vertical: - Boolean - Default: false
Enables the vertical.
duration: - Float - Default: 0.3
Sets the total duration time of the Action.
